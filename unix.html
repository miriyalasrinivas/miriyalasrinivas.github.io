Some Useful Commands:

  • grep for Tar files: tar -xzf filename.tar.gz --to-command='grep --label=$TAR_FILENAME -Hi "awk";true'
  • grep for Zip files: zipgrep select ./allgit/fadbbuildtasks/starterdb/lib/utplsql/utPlsql.zip
  • Command for Simple Encrypt and Decrypt Password:
  • echo "I AM GOOD" | openssl enc -base64
  • echo "SSBBTSBHT09ECg==" | openssl enc -base64 -d
  • Adding library path directories permanently:
  • [user@server1006@/home/user]$ cat /etc/ld.so.conf
  • include ld.so.conf.d/*.conf
